# 🏠 SafeStay — AI-Powered Housing Crisis Assessment

SafeStay is a production-ready, client-side web application designed to help tenants facing housing crises (evictions, habitability issues, lease disputes) by providing instant, caseworker-grade analysis using AI. 

SafeStay leverages advanced LLM reasoning to identify hidden flags, disqualification risks, and generate actionable step-by-step plans—without requiring the user to navigate a confusing legal bureaucracy or wait weeks for an initial consultation.

## ✨ Features

- **Instant Crisis Detection (Zero Latency):** A client-side heuristic engine parses input locally to detect severe crises (e.g., domestic violence, immediate danger) and immediately surfaces safety resources before any network requests are even made.
- **Adversarial Reasoning:** Powered by Anthropic's Claude (via direct integration or OpenRouter), the AI assesses situations adversarially—looking for what landlords might exploit and what tenants usually miss.
- **Single-Intake Workflow:** Replaces confusing chat-bot patterns with a clean, one-shot analysis, generating a 6-part actionable dossier.
- **Accessibility-First:** Fully WCAG-compliant design with ARIA labels, focus management, screen-reader optimizations, and high-contrast visuals.
- **Strict Data Privacy:** API keys and data are kept strictly in-memory or in local browser storage. No data is stored on external servers.
- **XSS & Security Hardened:** Robust output escaping, preventing prompt injection or malicious markup rendering.

## 🚀 Getting Started

SafeStay is entirely client-side and requires no backend server.

### Prerequisites
- A modern web browser
- An [OpenRouter](https://openrouter.ai/) or [Anthropic](https://console.anthropic.com/) API Key. *(Note: OpenRouter is recommended for local browser testing to avoid CORS issues.)*

### Installation
1. Clone this repository or download the source files.
2. Open `index.html` in any modern browser. No build steps required!
3. Click the ⚙️ gear icon in the top right to open settings.
4. Select your provider (`OpenRouter` recommended), input your API key, and hit Save.

## 🛠️ Architecture

SafeStay relies on a unidirectional data flow and modular agent pattern:
- `app.js` — The main controller. Handles the UI state machine, `AbortController` cancellation flows, and double-submit protections.
- `agents/stressCalibrator.js` — Tiered keyword analysis running locally. Identifies danger without latency.
- `agents/caseworkerReasoner.js` — Prompts the LLM with a highly constrained system prompt, grounding it in a local `knowledgeBase.js`. Forces structured JSON output mapping the crisis.
- `agents/llmClient.js` — A multi-provider abstraction layer.
- `renderer.js` — Safely parses the JSON output into an animated, print-friendly HTML dossier.

## 📖 Knowledge Base
The AI is grounded using a structured rule set covering:
- Eviction Notice Timelines
- Habitability and Repair Withholding
- Lease Violations
- Co-Tenant/Roommate Disputes
- Immigrant/Non-Citizen Housing Protections

## 🛡️ Responsible AI Compliance
- **Never Gives Legal Advice:** Explicitly avoids terms like "you qualify". Section 6 (Mandatory Human Referral) is hardcoded to render on *every* result.
- **Source Tracing:** Every action generated by the AI traces back to a specific rule ID in the knowledge base.
- **Privacy:** State is held entirely locally.
